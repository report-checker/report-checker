# Инструкция по использованию

## 1. Установка приложения из Tauri bundle

1. Перейдите в Releases: https://github.com/report-checker/report-checker/releases
2. Скачайте установочный файл под вашу ОС и архитектуру (`.msi`/`.exe` для Windows, `.dmg` для macOS, `.AppImage`/`.deb`/`.rpm` для Linux).
3. Запустите установщик и завершите установку.

## 2. Запуск установленного приложения

1. Откройте `report-checker` через меню приложений ОС.
2. Дождитесь загрузки главного окна приложения.

## 3. Загрузка PDF

1. Перетащите один или несколько PDF в левую панель.
2. Или нажмите область загрузки и выберите файлы вручную.
3. Нажмите `Запустить проверку`.

## 4. Просмотр результатов

- Левая панель: список файлов и сводка `OK / FAIL / WARN`.
- Центральная панель: предпросмотр PDF с подсветкой проблемных мест.
- Правая панель: дерево правил и вкладка `Текст ошибок`.

Как работать с результатами:

- Наведение на правило показывает подсветку в PDF.
- Клик по правилу закрепляет подсветку.
- Во вкладке `Текст ошибок` можно скопировать текст кнопкой `Copy`.

## 5. Настройка правил проверки

1. Откройте вкладку `Настройки` в центральной панели.
2. Измените параметры проверки.
3. Сохраните форму — конфигурация применяется сразу.

Файл конфигурации сохраняется рядом с исполняемым файлом Tauri:

- `checker.config.json`
